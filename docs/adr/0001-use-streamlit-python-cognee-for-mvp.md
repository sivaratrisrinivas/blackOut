# Use Streamlit, Python, and Cognee for the MVP

Status: Superseded by `0002-use-nextjs-frontend-with-python-workflow-api.md`.

BlackOut will use Streamlit with Python and the Cognee Python SDK for the hackathon MVP. This favors fast iteration, first-class Cognee integration, deterministic demo flows, and visible use of the memory lifecycle over the extra frontend and API plumbing of a Next.js plus FastAPI architecture.
